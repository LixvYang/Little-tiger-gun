### 项目运行

确保你下载了node，如果你没有下载请[点击](https://nodejs.org/)

```
#克隆到本地
git clone https://github.com/YLX621/Little-tiger-gun
#进入文件夹
cd Little-tiger-gun
#开启本地服务
node index.js
#开始玩游戏吧！
```
